# CI/CD Pipeline using Jenkins, Docker and Kubernetes

## Project Overview

This project demonstrates a complete CI/CD pipeline using GitHub, Jenkins, Docker and Kubernetes.

## Tools Used

* Git
* GitHub
* Jenkins
* Docker
* Kubernetes
* Linux

## Project Workflow

Developer
↓
GitHub
↓
Jenkins Pipeline
↓
Docker Image Build
↓
Kubernetes Deployment
↓
Application Access

kubectl get pods

## Outcome

Successfully automated application deployment using Jenkins, Docker and Kubernetes.
